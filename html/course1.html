Python для начинающих

Основы программирования с нуля

Автор: Иван Петров Длительность: 4 недели Уровень: Начинающий

Этот курс познакомит вас с основами программирования на Python. Вы научитесь писать простые программы, работать с данными и создавать свои первые проекты. Идеальный старт для тех, кто хочет освоить один из самых популярных языков программирования.

Программа курса

  • Введение в Python и установка среды разработки
  • Базовые конструкции языка: переменные, типы данных
  • Условные операторы и булева логика
  • Циклы и работа с итерациями
  • Функции и модули - организация кода
  • Работа с файлами и обработка исключений
  • Основы объектно-ориентированного программирования
  • Создание итогового проекта
